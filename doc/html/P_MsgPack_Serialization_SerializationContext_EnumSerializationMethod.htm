Property Value
Type: EnumSerializationMethod

            	The EnumSerializationMethod to determine default serialization strategy of enum types.
            
Remarks

            	A serialization strategy for specific member is determined as following:
            	If the member is marked with MessagePackEnumMemberAttribute and its value is not Default, then it will be used.Otherwise, if the enum type itself is marked with MessagePackEnumAttribute, then it will be used.Otherwise, the value of this property will be used.
            	Note that the default value of this property is ByName, it is not size efficient but tolerant to unexpected enum definition change.
